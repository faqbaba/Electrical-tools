# Electrical Engineering Tools (Python)

A collection of practical Python tools built for electrical engineering,
building management, and renewable energy applications.

## Tools Included
- Power Load Checker
- Inverter Sizing Tool (with report generation)
- Menu-Based Electrical Tools App

## Technologies
- Python 3
- File handling
- Modular programming

## Use Cases
- Electrical load estimation
- Inverter and battery sizing
- Building power management analysis
